About

In early 2020, the author of GeneticML, Valen Potloff, started a machine learning club at his high school in order to teach and discuss machine learning with his fellow students. A corresponding website was created for the club in order to assist in the demonstration of machine learning techniques. Traditionally, true machine learning education is reserved for late-undergraduate and graduate students, but this doesnâ€™t have to be the case. With the correct tools, machine learning can be taught at an approachable level to individuals with limited technical expertise. GeneticML provides the tools necessary for teaching a specific part of machine learning: genetic machine learning algorithms, which use the principles of genetics and natural selection in order to optimize machine learning models. This website contains literature regarding genetic machine learning algorithms, as well as interactive demonstrations of genetic machine learning algorithms in action.

About the Author:

Valen Potloff is currently an undergraduate student at the University of California, San Diego studying Data Science. Valen has over four years of experience creating and implementing machine learning algorithms for personal use and for professional business applications. As mentioned above, he also has experience in machine learning education, teaching high school students the basics of ML.
